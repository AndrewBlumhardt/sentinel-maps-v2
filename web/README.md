# Sentinel Maps V2

Azure Maps visualization for threat actor geolocation and real-time threat intelligence from Azure Sentinel.

## Features

- **Threat Actor Heatmap**: Visualize threat actors by country from TSV data
- **Threat Intelligence IPs**: Real-time IP threat data from Azure Log Analytics
  - Queries Azure Sentinel workspace
  - Geolocates IP addresses via Azure Maps
  - Displays aggregated bubbles by country
  - Interactive popups with threat details
